# System Health Monitoring Dashboard

A lightweight Flask-based monitoring dashboard designed to simulate basic network system monitoring.

--

# Features

- Real-time CPU usage monitoring
- Memory utilization tracking
- Disk space monitoring
- Auto refresh every 3 Seconds
- IST timezone display
- Dynamic alert color system 

--

# Tech Stack 

- Python
- Flask
- psutil
- HTML/CSS
- SSH-based Git deployment

--

## How to Run

1. Install dependencies:
   pip install flask psutil pytz

2. Run:
  python app.py

3. Open:
  http://127.0.0.1:5000/

--

## Use Case

This project demonstrates how system monitoring dashboard work in a small-scale security and network environments.
